# Privacy Settings Guide

## Overview
Control who can see your profile and what information is visible to others.

---

## Privacy Levels

### Profile Visibility

**Private**
- Only you can view your profile
- All other users are blocked
- Maximum privacy protection
- Recommended for: Users who want complete privacy

**Contacts Only**
- Only your contacts can view your profile
- Non-contacts are blocked
- Balanced privacy and connection
- Recommended for: Selective sharing with trusted connections

**Public**
- Anyone can view your profile
- Field-level privacy still applies
- Best for networking and collaboration
- Recommended for: Professional networking, team collaboration

---

## Field-Level Privacy

### Email Visibility

**Show Email (Enabled)**
- Your email address is visible to others
- Only applies if profile is Public or Contacts
- Useful for: Professional contact, collaboration

**Hide Email (Disabled)**
- Email address hidden from others
- You can always see your own email
- Recommended for: Privacy, reducing spam

### Activity Visibility

**Show Activity (Enabled)**
- Others can see:
  - When you joined
  - Last login time
- Builds trust and shows engagement
- Useful for: Team collaboration, active participation

**Hide Activity (Disabled)**
- Activity timestamps hidden from others
- You can always see your own activity
- Recommended for: Privacy-conscious users

---

## How Privacy Works

### Viewing Your Own Profile
- You always see all your information
- Privacy settings don't affect your own view
- All fields visible regardless of settings

### Others Viewing Your Profile

**If Profile is Private:**
- Access completely blocked
- Returns "Profile is private" error
- No information visible

**If Profile is Contacts Only:**
- Only contacts can view
- Non-contacts blocked
- Contact relationship required

**If Profile is Public:**
- Basic info always visible:
  - Display name
  - Bio
  - Avatar
- Email visible only if "Show Email" enabled
- Activity visible only if "Show Activity" enabled

## Privacy Scenarios

### Maximum Privacy
```
Profile Visibility: Private
Show Email: Disabled
Show Activity: Disabled
```
**Result:** Only you can see your profile

### Selective Sharing
```
Profile Visibility: Contacts Only
Show Email: Enabled
Show Activity: Enabled
```
**Result:** Contacts see full profile, others blocked

### Professional Networking
```
Profile Visibility: Public
Show Email: Enabled
Show Activity: Enabled
```
**Result:** Everyone sees full profile

### Public but Private Contact
```
Profile Visibility: Public
Show Email: Disabled
Show Activity: Enabled
```
**Result:** Profile visible, email hidden

## Changing Privacy Settings

### Update Profile Visibility
1. Go to Settings → Privacy
2. Select visibility level:
   - Private
   - Contacts Only
   - Public
3. Click "Save"
4. Changes take effect immediately

### Update Field Privacy
1. Go to Settings → Privacy
2. Toggle "Show Email" on/off
3. Toggle "Show Activity" on/off
4. Click "Save"
5. Changes apply instantly

## Common Questions

**Q: Can I hide my profile from specific users?**
A: Use "Contacts Only" and only add trusted users as contacts.

**Q: If I change to Private, can others still see my old activity?**
A: No, privacy changes apply immediately to all views.

**Q: Does hiding email prevent system emails?**
A: No, you still receive system emails. This only hides your email from other users.

**Q: Can I have different privacy for different users?**
A: Currently, privacy settings apply to all users uniformly. Use "Contacts Only" for selective sharing.

**Q: What happens to my data if I delete my account?**
A: Account deletion has a 30-day grace period, after which all data is permanently deleted.
