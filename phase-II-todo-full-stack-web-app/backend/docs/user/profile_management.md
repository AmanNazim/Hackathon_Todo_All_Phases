# Profile Management User Guide

## Getting Started

### Setting Up Your Profile

**Step 1: Add Display Name**
1. Navigate to Profile Settings
2. Enter your preferred display name (max 100 characters)
3. Click "Save"

**Step 2: Write Your Bio**
1. Add a brief description about yourself (max 500 characters)
2. Share your interests, role, or what you're working on
3. Click "Save"

**Step 3: Upload Avatar** (if available)
1. Click "Upload Avatar"
2. Select image (JPEG, PNG, or WEBP)
3. Maximum file size: 5MB
4. Image will be automatically optimized

## Profile Completion

Your profile completion percentage is based on:
- **Display Name**: 33%
- **Bio**: 33%
- **Avatar**: 34%

- Makes you more recognizable to others
- Builds trust in collaborative environments

**Change Avatar:**

## Privacy Settings

### Profile Visibility

**Private:**
- Only you can view your profile
- Recommended for maximum privacy

**Contacts Only:**
- Only your contacts can view your profile
- Good balance of privacy and connection

**Public:**
- Anyone can view your profile
- Best for networking and collaboration

### Email Visibility

**Show Email:**
- Others can see your email address
- Useful for professional networking

**Hide Email:**
- Email address hidden from others
- Recommended for privacy

### Activity Visibility

**Show Activity:**
- Others can see when you joined and last logged in
- Builds trust and shows engagement

**Hide Activity:**
- Activity timestamps hidden
- Recommended for privacy-conscious users

## Troubleshooting

**Profile not updating:**
- Check internet connection
- Refresh page
- Try again in a few moments

**Can't upload avatar:**
- Check file size (max 5MB)
- Verify file format (JPEG, PNG, WEBP)
- Try a different image

**Privacy settings not working:**
- Changes take effect immediately
- Log out and back in if issues persist
- Contact support if problem continues

**Notifications not received:**
- Check notification settings
- Verify email address is correct
- Check spam/junk folder
- Ensure notifications enabled in browser
