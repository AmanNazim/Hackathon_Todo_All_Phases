# Task Management User Guide

## Getting Started

### Creating Tasks

**Basic Task**:
1. Click "New Task" button
2. Enter title (required)
3. Add description (optional)
4. Set priority: low, medium, high, or urgent
5. Set due date (optional)
6. Click "Create"

**Quick Tips**:
- Use descriptive titles for easy searching
- Set realistic due dates
- Start with "todo" status

---

## Organizing Tasks

### Using Tags

**Add Tags**:
- Click task → "Add Tag"
- Type tag name (e.g., "work", "urgent")
- Press Enter

**Benefits**:
- Group related tasks
- Filter by multiple tags
- Track categories

**Best Practices**:
- Use consistent tag names
- Keep tags short (1-2 words)
- Limit to 3-5 tags per task

---

## Task Status Workflow

**Status Flow**:
```
todo → in_progress → review → done
       ↓
    blocked (from any status)
```

**Status Meanings**:
- **todo**: Not started
- **in_progress**: Currently working
- **review**: Awaiting review
- **done**: Completed
- **blocked**: Cannot proceed

---

## Searching Tasks

### Full-Text Search
- Searches title and description
- Case-insensitive
- Matches partial words

**Example**: Search "python" finds "Python programming" and "Learn python basics"

### Filtering

**By Status**: Show only todo, in_progress, or done tasks
**By Priority**: Filter high-priority items
**By Tags**: Show tasks with specific tags
**By Date**: Filter by due date range

**Combine Filters**: Use multiple filters together for precise results

---

## Batch Operations

### Bulk Updates

**Update Multiple Tasks**:
1. Select tasks (checkboxes)
2. Click "Batch Actions"
3. Choose operation:
   - Change status
   - Update priority
   - Add/remove tags
   - Delete tasks

**Use Cases**:
- Mark multiple tasks as done
- Add "urgent" tag to several tasks
- Archive completed projects

---

## Tips & Tricks

### Productivity
- Review tasks daily
- Use priority levels effectively
- Set realistic due dates
- Complete high-priority tasks first

### Organization
- Use tags for projects/categories
- Archive completed tasks regularly
- Keep task list under 50 active items

### Search
- Use specific keywords
- Combine search with filters
- Save common filter combinations

---

## Common Scenarios

**Weekly Planning**:
1. Filter by status: "todo"
2. Sort by priority
3. Set due dates for the week
4. Add "this-week" tag

**Project Management**:
1. Create tasks for project
2. Tag all with project name
3. Track progress by status
4. Use batch operations to update

**Daily Review**:
1. Check overdue tasks
2. Update task statuses
3. Complete finished tasks
4. Plan tomorrow's priorities

---

## Troubleshooting

**Can't find task**: Use search or check filters
**Task won't update**: Check internet connection
**Tags not showing**: Refresh page
**Slow performance**: Reduce active tasks or use filters
